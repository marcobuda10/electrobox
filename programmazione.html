Pin Tipo Funzione / Note
5V / 3.3VAlimentazioneFornisce tensione ai componenti esterni.
GNDMassaIl polo negativo del circuito (Ground).

Codice Base: Far lampeggiare un LED (Blink)

Il codice di partenza per verificare che tutto funzioni. Copialo nel tuo IDE:

void setup() {
  // Imposta il pin 13 come uscita digitale
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH); // Accende il LED (Invia 5V)
  delay(1000);           // Aspetta 1 secondo (1000 millisecondi)
  digitalWrite(13, LOW);  // Spegne il LED (Invia 0V)
  delay(1000);           // Aspetta un altro secondo
}
